# csv_data_explorer

CSV Data Explorer è un progetto sviluppato in Python per esplorare e analizzare file CSV.

L'obiettivo del progetto è imparare le basi della Data Analysis utilizzando Pandas, NumPy e Matplotlib.

---

## 🎯 Obiettivi

- Caricare un dataset CSV
- Analizzare la struttura dei dati
- Individuare valori mancanti
- Calcolare statistiche descrittive
- Visualizzare informazioni utili
- Esportare dati filtrati

---

## 🛠 Tecnologie

- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

---
